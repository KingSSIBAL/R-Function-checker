# Find positions where vectors differ

Find positions where vectors differ

## Usage

``` r
.cpp_find_differences(v1, v2, tolerance = 1e-10, max_diffs = 10L)
```

## Arguments

- v1:

  First numeric vector

- v2:

  Second numeric vector

- tolerance:

  Comparison tolerance

- max_diffs:

  Maximum differences to return

## Value

Integer vector of differing positions (1-based)
